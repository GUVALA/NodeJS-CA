# Global Kitchen Recipe API

A RESTful API built with Node.js, Express.js, and MongoDB for managing recipes.

## Features
- Create recipes
- Read recipes
- Update recipes
- Delete recipes
- Filter by category
- MVC + Service Layer architecture

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- nodemon
